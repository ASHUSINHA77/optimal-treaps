# Optimal Treaps Visualizer

An interactive, web-based algorithmic visualization tool that demonstrates the mechanics of an **Optimal Treap** (Randomized Search Tree). This application bridges the properties of a Binary Search Tree (BST) and a Heap structure to maintain balanced search steps.

## 🚀 Features
* **Dynamic Tree Rendering:** Watch the tree structure self-correct in real-time as keys are manipulated.
* **Key Operations:**
  * **Insert Key:** Adds elements while assigning randomized priorities to maintain heap properties via tree rotations.
  * **Delete Key:** Seamlessly removes nodes and restructures the remaining tree.
  * **Search Key:** Visually traces the path down the tree to locate target integers.
* **Live State Analytics:** Tracks real-time tree statistics including total node **Size** and overall tree **Height** to evaluate performance.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript / React (Canvas-based rendering)
* **Hosting Environment:** Replit Dev Environment
